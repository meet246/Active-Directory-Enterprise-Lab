# PowerShell Automation

This folder contains the PowerShell scripts used to automate Active Directory administration throughout the Enterprise Active Directory Home Lab.

---

## Create-Users.ps1

Automates enterprise onboarding by:

- Creating Organizational Units
- Creating Security Groups
- Importing users from a CSV file
- Provisioning Active Directory user accounts
- Assigning users to department security groups

---

## Configure-HR-Login-GPO.ps1

Automates Group Policy configuration by:

- Creating an HR login banner
- Configuring registry values
- Linking the GPO to the HR Organizational Unit
- Applying enterprise security policy
